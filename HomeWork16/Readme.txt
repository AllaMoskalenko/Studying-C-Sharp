Создать классы для определения фигур: круг, прямоугольник, правильный многоугольник, звезда и агрегация:
у каждой из фигур есть свои поля:
у круга - радиус;
у прямоугольника - ширина и длинна;
у многоугольника - радиус и количество вершин
у звезды - внутренний/внешний радиусы и количество вершин
у агрегации - набор фигур

также у каждой из фигур есть общие поля и методы, которые определены в классе-родителе(желательно абстрактный):
- координаты Х и У;
- цвет;
- отрисовать();
- переместить();

Программа должна поддерживать следующие действия:
1. создание/удаление объектов
2. выбор одного или нескольких объектов
3. перемещение выделенных фигур мышкой
4. поменять цвет и расположение фигуры
5. агрегировать фигуры (т.е. создать новую фигуру из фигур, при чем фигуры, составляющие агрегат удаляются) или разбить фигуру на составляющие (если это была агрегация)
6. сохранить текущее состояние всех фигур в файл
7. загрузить сохраненное состояние фигур из файла
для отрисовки на pictureBox воспользуйся Bitmap
