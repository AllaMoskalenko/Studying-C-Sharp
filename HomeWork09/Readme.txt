есть файл с известным названием, в котором содержится информация о какой-то звездной системе.
в каждой строке файла идет информация об объекте системы:
если это звезда, то строка состоит из названия и температуры;
если это планета, то строка состоит из названия планеты и последовательности названий спутников через пробел (спутников может и не быть)
например 
SolarSystem.txt
{
Sun 5700  //звезда
Mercury   //планета без спутников
Venus
Earth Moon  //планета со спутниками
Mars Phobos Deimos
}
(для наших задач будет всего три вида объектов: звезда, планета и спутник)

Создать классы SpaceObj, Star, Planet, Moon, в которых определить поля и методы как на картинке.
Эти свойства классов являются минимально-необходимыми (т.е. их можно дополнять).
SpaceObj в данном случае будет родительским классом, от которого наследуются классы Star, Planet, Moon.(что также показано на картинке)

Задача:
1. Считать из файла☝️ информацию о звездной системе.
2. Предоставить возможность пользователю отдельно выбрать любой объект системы и посмотреть о нем информацию. (подразумеваю, что после прочтения файла образуется 
список объектов, из которых и можно выбирать)
3. Посчитать количество звезд, планет и спутников в системе.
4. вывести название планеты с наибольшим количеством спутников.
