написать симулятор графа (математической структуры)
оно будет состоять из трех больших частей
———ЧАСТЬ1———
создать класс Graph, в котором должны быть реализованы методы:
- добавить/убрать вершину
- добавить/убрать ребро
- отобразить граф на экране (в виде вершин и ребер)

Дать возможность пользователю:
- создавать и удалять вершины
- создавать и удалять ребра между вершинами (при чем дать возможность задавать вес ребра)
- показать граф виде матрицы смежности, матрицы инцидентности и списка смежности (использовать extension(расширение)!)
- сохранить файл с расширением .graphHW19 с помощью "FileDialog"
- открыть ".graphHW19" с помощью "DragAndDrop"
- открыть ".graphHW19" с помощь "FileDialog"

———ЧАСТЬ2———
в класс Graph добавить методы:
- поиск в ширину/в глубину
- поиск пути между двумя вершинами

Дать возможность пользователю:
- обойти граф поиском в глубину и вывести последовательность обхода вершин
- обойти граф поиском в ширину и вывести последовательность обхода вершин
- найти путь между двумя вершинами алгоритмом Дейкстры

———ЧАСТЬ3———

- разукрасить граф

- разукрасить вершины графа (решить задачу о раскраске графа)
