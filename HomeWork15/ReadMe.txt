Создать программу для конструирования ракет:

Создать три интерфейса:
1) определяющий корпус ракеты (IBody)
2) определяющий топливный бак ракеты (IFuelReservoir)
3) определяющий двигатель ракеты (IEngine)
От каждого из интерфейсов наследуется по три класса, определяющие подвиды деталей ракет
И итоговый класс (Rocket), содержащий элементы ракеты.

Требования: 
1) сохранение/запись ракет в файл (название файла заранее известно)
2) работа с несколькими формами (основная форма и формы для редакции/создания ракет)
3) !!!не использовать DataGridView!!!

Рекомендации:
1) для списка всех ракет лучше использовать не массив, а структуру List<>. Она более гибкая и удобнее в плане добавления 
новых элементов.

Задачи: 
1) Считать из файла список ракет (если файла нету или он пустой - сообщить об этом)
2) Сохранить текущее состояние списка всех ракет
3) Дать возможность отсортировать ракеты по стоимости/массе/названию (по выбору пользователя)
4) Дать возможность добавления к существующему списку новых ракет (при добавлении открывается форма-редактор, где юзер может выбрать, из каких частей будет 
состоять ракета)
5) Дать возможность редактирования выбранной ракеты из списка всех ракет (при редактировании открывается форма-редактор, где юзер может выбрать, из каких частей 
будет состоять ракета). При чем когда откроется окно редакции - элементы ракеты уже выбраны.
