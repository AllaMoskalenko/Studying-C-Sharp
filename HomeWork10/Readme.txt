Cоздать класс "Circle", в котором будут следующие поля:
int x,y;
int diametr;
и методы:
move(int dx, int dy);
draw(Graphics g);
Задача: 
1. Создать массив из n "Circl-ов" (по нажатию на кнопку)
2. отобразить их всех на экране
3. предусмотреть кнопку для перемещения выбраного круга (имею ввиду, что из перечня(ComboBox) можно выбрать круг, а потом по нажатию кнопки переместить его на dx вправо и на dy вниз)

рекомендуемые функции:
1) g.DrawEllips(pen, x, y, width, height); - для отображения эллипса (чтобы нарисовать круг - параметры  width и height равны)
2) при "перемещении" круга происходит следущее:
   1. все стирается с экрана;
   2. перемещаем выбраный круг на dx и dy с помощью метода move(dx, dy);
   3. отрисовываем ВСЕ круги заново с уже перемещенным выделенным кругом.
